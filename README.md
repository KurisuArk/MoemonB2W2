## What Does This Patch Do?

This patch modifies **Pokémon Black 2 / White 2** with the following features:

### Core Changes

- Replaces all Pokémon sprites with **Moémon sprites**
- Makes **all Pokémon from Generations 1–5 obtainable**
- Replaces all mentions of "Pokemon" in text with "Moemon"
- Unlocks **Easy Mode and Challenge Mode** from the beginning  
  *(Simply play until you receive your starter.)*
- Unlocks **Black City and White Forest** in both versions
- Unlocks **Iron Chamber and Iceberg Chamber** in both versions
- Unlocks **National Dex from the start**
- Shows the highest IV a Moémon has in the **Characteristics** text

### Quality-of-Life Improvements

- Replaces trade evolution methods and adds extra methods for some Pokémon
- Adds evolution items to **Shopping Mall Nine**
- Faster HP bar speed (especially noticeable at lower levels)
- Changes the title screen and intro
- Increased female ratio
- Increased shiny rate to **1/257**

## Disclaimer

- Anything not mentioned remains the same as base Pokémon Black 2 / White 2.
- The sprites were originally made for GBA.  
  For that reason, back sprites do not have full-body artwork.

## Patching on PC

1. Download **Delta Patcher Lite** (from the provided Google Drive folder).
2. Open Delta Patcher Lite (or another compatible DS patcher).
3. Under **Original file**, select a Black 2 or White 2 ROM  
   (The patch should be compatible with any B2/W2 ROM.)
4. Under **XDelta patch**, select the patch file  
   (It contains `[Patch]` in the name and has a `.xdelta` extension.)
5. Click **Apply patch**.
6. Done — you're ready to play.

---

## Patching on Mobile (Android)

1. Download and install **UniPatcher**.
2. Open UniPatcher.
3. In **Patch file**, select the `.xdelta` patch file  
   (Contains `[Patch]` in the name.)
4. In **ROM file**, select a Black 2 or White 2 ROM.
5. In **Output file**, choose a folder where the patched ROM will be saved.
6. Tap the red button and wait.
7. Done — you're ready to play.
